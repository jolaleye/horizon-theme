Installation

  1. Open the extensions sidebar in VS Code.
  2. Search for Horizon Theme .
  3. Click install .
  4. Open the Command Palette with Ctrl+Shift+P or ⇧⌘P .
  5. Select Preferences: Color Theme and choose a Horizon variant.
  6. Enjoy!
